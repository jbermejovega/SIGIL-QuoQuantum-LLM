# n8n → PYDANTIKA → QUAZRIS8 Migration Glue V1

Legacy n8n payloads are accepted only as source material for typed conversion.

```text
n8n payload
-> provenance check
-> PYDANTIKA validation
-> QUAZRIS8 gates
-> MCP request or typed obstruction
-> external witness verification
```

A workflow success flag is never treated as repository truth, proof, or certificate.
